# Arknights Codex Pets v1.0.0

The first public release contains seven Codex V2 animated pets:

- Doctor Doctor (`doctor-doctor`)
- Mimiko (`mimiko`)
- Kal'tsit-tan (`kaltsit-tan`)
- Vivimi (`weiweimei`)
- Bright-Eyed SilverAsh (`liangyan-yinhui`)
- Wandering Exusiai (`xianyou-nengtianshi`)
- Texas (`jiemen-dekesasi`)

Each package contains `pet.json` and a validated `1536 × 2288` RGBA `spritesheet.webp`. The all-in-one archive also includes macOS/Linux and Windows installers, English documentation, character cards, and full animation contact sheets.

All seven atlases pass the Hatch Pet V2 structural validator with zero errors and zero warnings.
